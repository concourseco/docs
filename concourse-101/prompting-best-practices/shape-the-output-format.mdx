Tell Concourse how you plan to use the answer.

- **Tone and density** – "Keep it under 5 bullets", "Explain it like I am new to the business", "Assume an expert finance audience".
- **Structure** – "Use a table with columns: metric, current period, prior period, variance, variance %", "Group by department, then sort by variance descending".
